■概要
箱の中をオブジェクトが跳ね返っているだけの観賞用途のシェーダーです。
パラメータからある程度外観を調整することができます。

表示オブジェクトは以下を対応しています。
・Sphere
・Box
・Torus（ドーナツ）
・Octahedron（八面体）
・LightBulb（電球）

本シェーダーはレイマーチングで実装されています。
距離関数を追加することで表示されるオブジェクトを追加することもできます（頑張れば）。


■パラーメータ
(1)Display Object
　表示されるオブジェクトの切り替え
　
(2)Light
　・Intensity
　　　光の強さ
　・Bloom
　　　Bloomエフェクトの強さ

(3)Bound Flash
　・Parent
　　ここで指定したオブジェクトが壁に衝突したときに全てのオブジェクトが光る
　　（個別で光らせようと試みましたが失敗しました）
　・Intensity
　　壁に衝突したときの光の強さ
　・Threshold
　　壁とオブジェクトの距離がThreshold以下になったとき衝突したと判定する
　　
(4)Enviroment
　・Phantom Mode
　　Onにするとオブジェクトが半透明になる
　・Color Change Speed
　　オブジェクトの色の変化スピード
　・Saturation
　　オブジェクトの彩度
　・Reflection Box Size
　　反射する壁のサイズ
　　（xyzを全て同じ数値に設定すると同じパターンでしかバウンドしないため初期値では若干ズラしています）

(5)Object Setting
　・Seed
　　各オブジェクトの位置をバラけさせるため指定した値だけ経過時間をプラスする
　・Speed
　　各オブジェクトの移動スピード
　・Size
　　各オブジェクトのサイズ
